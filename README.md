# Análisis de Señales y Transformada de Fourier

## Descripción

Este proyecto tiene como objetivo analizar señales en el dominio del tiempo y de la frecuencia utilizando Python y la Transformada Rápida de Fourier (FFT).

Se implementaron diferentes señales básicas, como la señal senoidal, el pulso rectangular y la función escalón. Posteriormente, se calculó su Transformada de Fourier para observar su comportamiento en el dominio de la frecuencia.

## Objetivos

* Representar señales en el dominio del tiempo.
* Aplicar la Transformada de Fourier mediante FFT.
* Visualizar la magnitud y fase del espectro de frecuencia.
* Analizar las propiedades de linealidad y desplazamiento temporal.
* Comparar la representación de las señales en ambos dominios.

## Herramientas utilizadas

* Python 3
* NumPy
* Matplotlib

## Archivos del proyecto

* Fourier.py: código principal de la simulación.
* README.md: descripción del proyecto.
* Gráficas generadas durante el análisis.

## Resultados

Los resultados muestran cómo cada señal posee una representación diferente en el dominio de la frecuencia. Además, se verifican propiedades importantes de la Transformada de Fourier, como la linealidad y el desplazamiento temporal.
